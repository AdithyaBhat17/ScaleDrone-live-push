You can push a notification from Terminal using cURL: 
        
 working code

        
            curl -H "Content-Type: application/json" \
-X POST \
-d '{"msg":"Hello!"
}' \
https://api2.scaledrone.com/2AbrIR7BniItM6um/notifications/publish
         

         
 code under testing

         
            curl -H "Content-Type: application/json" \
-X POST \
-d '{"msg":"Hello!",
        "title":"title!"
}' \
https://api2.scaledrone.com/2AbrIR7BniItM6um/notifications/publish
