You can push a notification from Terminal using cURL: 
        
 working code Image is hardcoded

        
            curl -H "Content-Type: application/json" \
-X POST \
-d '{"msg":"Hello!"
}' \
https://api2.scaledrone.com/2AbrIR7BniItM6um/notifications/publish
         

         
 Send with an image

         
            curl -H "Content-Type: application/json" \
-X POST \
-d '{"msg":"Hello!", "url":" https://pbs.twimg.com/profile_images/832895558318694400/UH6Lt_k2.jpg"}' \
https://api2.scaledrone.com/2AbrIR7BniItM6um/notifications/publish
         
         
Send with title body etc

         
curl -H "Content-Type: application/json" \
-X POST \
-d '{"msg":"Hello!", "title":" HIIIII"}' \
https://api2.scaledrone.com/2AbrIR7BniItM6um/notifications/publish
